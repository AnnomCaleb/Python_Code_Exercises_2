Patient Status Check System

This project is a simple Python program that determines what action a patient should take based on their status. It uses user input and conditional statements to simulate a basic hospital decision system.

I built this project to practice working with inputs, data types, and conditional logic in Python.

Features
Accepts patient details (name, age, and status)
Uses conditional statements (if, elif, else)
Handles input case sensitivity using .lower()
Provides different outputs based on patient status
Tech Stack
Python
How it works

The program asks the user to enter:

Their name
Their age
Their status (new or old)

Based on the status entered:

New patients are allowed to see a doctor
Old patients are directed to get a new appointment
Any other input is treated as not being in the system
Why I built this

I wanted to improve my understanding of how programs make decisions using conditional statements. This project also helped me practice handling and validating user input.

Getting Started
